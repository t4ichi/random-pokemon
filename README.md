# TypeScript+Next.jsで作るポケモンジェネレーター

TypeScriptとNext.jsで作った簡単なアプリケーション

## 使い方

セットアップ
```bash
yarn create next-app --example with-typescript random-pokemon
```

アプリケーションの起動
```bash
cd random-pokemon
yarn dev
```
